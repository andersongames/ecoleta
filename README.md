# Projeto Ecoleta
 
Projeto **Ecoleta**, desenvolvido na Next Level Week da Rocketseat. :rocket:

<a href="https://www.figma.com/file/Byw4X5etg8VCmezueyhzkC/Ecoleta-(Starter)?node-id=136%3A546" target="_blank">Figma</a>
